Contenido

Propósito del sistema

Facilitar el registro, consulta y gestión de cursos y notas académicas.

Requisitos

Tener instalado Python 3.x.

Ejecutar el programa desde la consola:

python gestor_notas.py


Uso del menú

Se muestran 13 opciones numeradas.

El usuario selecciona la opción deseada ingresando su número.

Ejemplos

Ingrese el nombre del curso: Matemáticas
Ingrese la nota: 85
✅ Curso registrado con éxito.


Ejemplo de promedio

Promedio general: 87.5


Simulación de cola

Ingrese curso para revisión (escriba 'fin' para terminar):
> Matemáticas
> Historia
> fin
Procesando solicitudes:
Revisando: Matemáticas
Revisando: Historia

Preguntas frecuentes

¿Qué pasa si ingreso una nota mayor a 100?
→ El sistema no la acepta.

¿Se pueden guardar los datos al salir?
→ No, el programa es temporal (solo en ejecución actual).
