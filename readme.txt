Readme

Descripción General
Este proyecto implementa un sistema de registro académico en consola, que permite gestionar cursos y notas de un estudiante.  
El sistema utiliza listas, pilas, colas, algoritmos de ordenamiento y búsqueda de acuerdo con los principios de programación estructurada en Python.

Funcionalidades Principales
- Registrar cursos y notas con validación de rango (0-100)
- Mostrar cursos y notas registradas
- Calcular promedio general
- Contar cursos aprobados y reprobados
- Buscar cursos (búsqueda lineal y binaria)
- Actualizar notas
- Eliminar cursos con confirmación
- Ordenar cursos por nota (Burbuja)
- Ordenar cursos por nombre (Inserción)
- Simular una cola de solicitudes de revisión
- Mostrar historial de cambios(pila)

Tecnologías Utilizadas
- Lenguaje: Python
- Paradigma: Programación estructurada
- Sin librerías externas ni bases de datos


Reflexión Personal

Este proyecto me permitió aplicar de forma práctica los conocimientos sobre listas, pilas, colas y algoritmos de ordenamiento y búsqueda**.  
Aprendí la importancia de estructurar el código por funciones y mantener organizados los datos relacionados.  
El mayor reto fue lograr que las listas de cursos y notas se mantuvieran sincronizadas al ordenar o modificar valores.  
En general, fue una experiencia útil para fortalecer mi lógica de programación y pensamiento algorítmico.
